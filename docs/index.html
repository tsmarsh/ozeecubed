Getting Started

Quick Start

  1. Download and extract the binary for your platform
  2. Connect a microphone or audio input device
  3. Run ozeecubed (or ozeecubed.exe on Windows)
  4. Make some noise and watch the waveform appear!
  5. Use keyboard shortcuts to adjust time/voltage scales

The oscilloscope will automatically detect and use your system's default audio input device. If no audio device is detected, it will display a test signal (440 Hz sine wave).

Build from Source

Want to build from source? It's easy with Rust:

git clone https://github.com/tsmarsh/ozeecubed.git
cd ozeecubed
cargo build --release
./target/release/ozeecubed

Requires Rust 1.70+ and platform-specific audio dependencies (ALSA on Linux, CoreAudio on macOS).
